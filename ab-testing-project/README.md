# 📊 Análisis de Experimento A/B
## 🎯 Objetivo

Evaluar el impacto de un nuevo sistema de recomendación en el comportamiento de usuarios.

## ❗ Problema

Determinar si los cambios implementados generaban mejoras reales en métricas clave de conversión.

## ⚙️ Proceso

Limpieza y validación de datos

Análisis exploratorio de usuarios

Evaluación de grupos control vs prueba

Análisis estadístico

Visualización de resultados

## 📈 Resultados / Insights

Se detectó un desbalance en la asignación de usuarios entre grupos

Esto afectaba la validez del experimento

Las métricas no eran completamente confiables

## ✅ Conclusión

El experimento no permite obtener conclusiones definitivas debido a problemas en el diseño. Es necesario corregir la asignación de grupos antes de tomar decisiones basadas en estos resultados.

## 🛠️ Tecnologías utilizadas

Python | Pandas | NumPy | A/B Testing | Visualización de datos
